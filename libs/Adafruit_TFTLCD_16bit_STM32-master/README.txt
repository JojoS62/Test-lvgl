This is a library for 16bit parallel driven TFT displays, ported from the Adafruit library. https://github.com/adafruit/TFTLCD-Library

- Place this library folder your <arduinosketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE.

- Also requires the Adafruit_GFX library for Arduino. https://github.com/adafruit/Adafruit-GFX-Library
